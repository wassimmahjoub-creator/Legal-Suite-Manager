Remplace tous les "Aucune donnée / لا توجد" plats par des empty states pédagogiques actionnables.

COMPOSANT
Crée src/components/ui/EmptyState.tsx :

interface EmptyStateProps {
  illustration?: ReactNode; // SVG ou icône
  title: string;
  description: string;
  primaryAction?: { label: string; onClick: () => void; icon?: ReactNode };
  secondaryAction?: { label: string; onClick: () => void; icon?: ReactNode };
  helpLink?: { label: string; href: string };
}

Layout vertical centré, illustration en haut (160-200px), titre h3, description text-muted, boutons côte à côte, lien d'aide en bas en text-sm underline.

ILLUSTRATIONS
Crée src/components/illustrations/ avec un SVG simple par contexte :
- EmptyCases.tsx (dossiers de justice empilés)
- EmptyClients.tsx (silhouettes vides)
- EmptyInvoices.tsx (facture)
- EmptyCalendar.tsx (calendrier vide)
- EmptyDocuments.tsx (dossier vide)

Garde-les minimalistes, monochrome avec la couleur dorée du brand, sans librairie externe.
