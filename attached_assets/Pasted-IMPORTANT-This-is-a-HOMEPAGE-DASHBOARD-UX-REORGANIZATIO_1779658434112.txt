Goal:
Transform the homepage from a generic SaaS dashboard into a daily operational cockpit for Tunisian lawyers.

The homepage must answer one question immediately:

"شنوّة يلزمني نعمل اليوم؟"

The lawyer must instantly see:
- today’s hearings
- urgent legal deadlines
- tasks to complete
- files needing attention
- recent activity
- quick actions

Financial metrics should remain available but must not dominate the homepage.

1) CORE UX PRINCIPLE
====================================================

The current homepage gives too much visual weight to generic KPIs and finance.

Reorganize the page around the lawyer’s real morning priorities:

1. جلسات اليوم
2. آجال قانونية قريبة
3. المهام العاجلة
4. التنبيهات المهمة
5. آخر النشاطات
6. آخر الملفات
7. ملخص مالي صغير

The dashboard must feel:
- operational
- urgent when needed
- calm when nothing is critical
- compact
- useful
- professional

NOT:
- generic SaaS analytics
- finance-first
- empty cards
- oversized KPI layout

====================================================
3) PRIMARY SECTION: اليوم
====================================================

Create the main top section:

اليوم

This must be the first major section on the page.

It contains three cards side-by-side on desktop, stacked on mobile:

1. جلسات اليوم
2. آجال قريبة
3. المهام العاجلة

====================================================
3A) جلسات اليوم
====================================================

This card is the highest priority.

Display:
- time
- court
- chamber/circuit
- file name
- client
- responsible lawyer

Example:
09:00 — المحكمة الابتدائية بتونس
قضية شركة التركي
الحريف: شركة التركي

If no hearings today:
Do NOT waste a large empty card.

Instead show:
"لا توجد جلسات اليوم"
and below:
"الجلسة القادمة: [date] — [file]"

Add button:
+ حدث جديد

====================================================
3B) آجال قريبة
====================================================

Display legal deadlines.

Examples:
- آخر أجل للاستئناف
- أجل التعقيب
- أجل الرد
- أجل الإعلام

Each item must show:
- file
- deadline type
- remaining days
- urgency status

Urgency rules:
- today / overdue = red
- within 3 days = red
- within 7 days = amber
- later = neutral

Example:
آخر أجل للاستئناف — باقي 3 أيام
ملف: قضية عقار الزهراء

====================================================
3C) المهام العاجلة
====================================================

Display:
- overdue tasks
- tasks due today
- important tasks

Each task:

- title
- linked file
- priority
- responsible person

Add button:
+ مهمة جديدة
